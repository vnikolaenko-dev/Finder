# Найдсь (Finder)
Найдись – уникальная платформа, позволяющая школьникам и студентам создавать проекты, искать единомышленников, прокачивать свои soft и hard skills в современной, геймифицированной форме.
# О проекте
Проект разрабатываю в свободное от учебы время, если тебя заинтересовала идея - пиши в тг (см шапку профиля)
