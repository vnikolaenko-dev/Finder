# Найдсь (Finder)
Найдись – уникальная платформа, позволяющая школьникам и студентам создавать проекты, искать единомышленников, прокачивать свои soft и hard skills в современной, геймифицированной форме.
Найдись объединяет школьников и студентов для воплощения идей в жизнь! Общайтесь, делитесь успехами проекта и своими собственными достижениями, зарабатывайте очки и получайте награды! Чем больше очков, тем выше рейтинг проекта.
Участники проекта найдут не только единомышленников, но и получат приглашение на стажировку! Участников с наибольшим рейтингом по результатам сезона пригласят компании-партнеры!
Крупные компании получают возможность публиковать свои авторские кейсы и проекты для участников платформы!
